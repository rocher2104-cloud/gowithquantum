Design Tokens

CSS custom properties that power the entire system. Reference these variables in all components — never hardcode values.

/* Paste into :root {} */ /* Backgrounds */ --color-bg-page: #F5F5F5; --color-bg-panel: #FFFFFF; --color-bg-sidebar: #111111; --color-bg-selected: #F3F4F6; --color-bg-hover: #F9FAFB; /* Text */ --color-text-primary: #111111; --color-text-secondary: #6B7280; --color-text-tertiary: #9CA3AF; /* Borders */ --color-border: #E5E7EB; --color-border-strong: #D1D5DB; /* Actions */ --color-cta: #000000; --color-accent-blue: #3B82F6; --color-accent-blue-bg: #EFF6FF; /* Status */ --color-success: #22C55E; --color-warning: #EAB308; --color-error: #EF4444;

Spacing

Strict 8px grid. All margins, padding, and gaps use these values exclusively. Never use arbitrary pixel values.
